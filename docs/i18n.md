# Localization (i18n)

- **Libraries**: `i18next`, `react-i18next`, `expo-localization`.
- **Config**: Setup in `src/i18n/index.ts`.
- **Translations**: JSON files in `src/i18n/locales/`.
- **Usage**: React hook `useTranslation()`.
- **Rule**:
  - No hardcoded text in UI.
  - Fallback lang is `en`.
  - Keep keys semantic and nested by screen/component.
